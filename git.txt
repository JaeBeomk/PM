git 관리 방법

- 프로젝트 가져오기 
git clone https://github.com/JaeBeomk/PMC.git

- branch(작업공간) 생성
git checkout -b "[name]"

- branch(작업공간) 전환
git checkout "[name]"

- branch 생성 확인
git branch

# 실제 작업스텝
$ git checkout DEV
$ git checkout -b ["name"]
----- work ------
$ git add .
$ git commit -m "[author] : xxx , [message] : xxxx"
$ git checkout DEV
$ git merge ["name"]
