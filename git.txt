git 관리 방법

- 프로젝트 가져오기 
git clone https://github.com/JaeBeomk/PMC.git

- branch(작업공간) 생성
git checkout -b "[name]"

- branch(작업공간) 전환
git checkout "[name]"

- branch 생성 확인
git branch




git remote -v -> 내 파일들이 올라갈 url(주소)확인

git push origin feature/kangminji -> 푸쉬! 올리기!

# 실제 작업스텝
$ git checkout DEV
$ git checkout -b ["name"]  -> branch 생성 후 즉시 적용
$ git branch -> 내 브랜치 확인
$ git merge DEV -> DEV branch 내용 가져오기
